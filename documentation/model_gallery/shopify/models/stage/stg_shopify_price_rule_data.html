üìÉ 1. Table Summary


The table is about Shopify price rules. It contains details of discount configurations. Each rule has an ID, creation date, and expiration date. Rules specify customer eligibility, discount type, and value. They can target specific items or all products. Additional fields set prerequisites like minimum purchase amounts. The table allows for flexible discount creation and management in the Shopify platform.
